# Implementer Output Template

```md
【角色结论】
- 用一句话总结本次实现是否严格按方案完成。

【交付物】
- 【修改文件】
  - 文件 1:
  - 文件 2:
- 【具体改动】
  - 改动点 1:
  - 改动点 2:
- 【新增代码】
  - 新增逻辑 1:
  - 新增逻辑 2:

【约束】
- 审核只能基于批准方案进行
- 方案外改动默认视为违规

【校验标准】
- 实现与方案一致
- 仅修改批准文件
- 无无关优化混入

【禁止事项】
- 不重设计
- 不新增功能
- 不做顺手清理

【交接给下一个角色】
- Next role: reviewer
- allowed inputs: 原始需求 + solution handoff + 改动代码 + 本 handoff
- non-goals: 重写需求或架构
```
